# 🛒 Family Cart


Family Cart — це зручний веб-застосунок для спільного керування списком покупок для всієї сім'ї. Завдяки синхронізації даних у реальному часі, усі члени родини бачать актуальний список продуктів на своїх пристроях, що робить процес шопінгу організованим та комфортним.

⚡ Основні можливості
👥 Спільний доступ: Створюйте, редагуйте та видаляйте товари — зміни миттєво відображаються у всіх учасників.

☁️ Хмарне зберігання: Ваші списки завжди в безпеці та доступні з будь-якого пристрою завдяки Firebase.

📱 Адаптивний дизайн: Зручний інтерфейс, оптимізований як для десктопів, так і для смартфонів.

✨ Інтуїтивний інтерфейс: Чистий та мінімалістичний дизайн для швидкої взаємодії.
